# Specifica Etichetta QR Code вҖ” KYKOS

## Panoramica

Stampare un'etichetta adesiva (50Г—30 mm o 50Г—40 mm) da applicare all'oggetto consegnato presso l'ente.
L'etichetta contiene un QR code che, in fase di ritiro, consente all'operatore di verificare l'identitГ  dell'oggetto.

---

## Configurazione

L'ente puГІ configurare la stampa etichetta dalla pagina `/operator/organization`:

| Opzione | Descrizione |
|---------|-------------|
| **Stampa etichetta** | Flag yes/no per abilitare la stampa |
| **Formato** | `50x30` (standard) o `50x40` (grande) |

## Contenuto QR Code

**Stringa per QR di consegna (deliver):**
```
kykos:object:deliver:{requestId}:{donorId}
```

**Stringa per QR di ritiro (pickup):**
```
kykos:object:pickup:{requestId}:{recipientId}
```

---

## Flusso Operativo

### Flusso Consegna (Deposit)

```
1. Operatore scansiona QR oggetto вҶ’ /operator/deposit/[requestId]
2. Operatore registra posizione deposito
3. Se printLabel = true:
   вҶ’ Mostra dialog opzionale stampa etichetta
   вҶ’ L'operatore puГІ Stampa o Salta
4. Redirect a /operator/scan-qr?success=deposit
```

### Flusso Ritiro (Pickup)

```
1. Operatore scansiona QR ritiro вҶ’ /operator/pickup/[requestId]
2. Sistema mostra info oggetto e posizione
3. Se printLabel = true:
   вҶ’ Mostra sezione verifica QR (scansiona QR sull'etichetta)
   вҶ’ L'operatore puГІ scansionare o procedere senza verifica
4. Operatore conferma ritiro completato
5. Redirect a /operator/scan-qr?success=pickup
```
